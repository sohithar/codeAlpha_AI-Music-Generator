## 🐛 Troubleshooting

### Website doesn't load?
```
Solution:
1. Make sure you're opening the right file: index.html
2. Check file is in: c:\Users\DELL\Desktop\sriku - codealpha 1\
3. Try using a different browser (Chrome, Firefox, Safari, Edge)
4. Check browser console for errors (F12)
```

### Styles look wrong?
```
Solution:
1. Hard refresh: Ctrl+Shift+R (or Cmd+Shift+R on Mac)
2. Clear browser cache
3. Try in different browser
4. Check that style.css is in same folder as index.html
```

### Animations too slow?
```
Solution:
1. Close other programs using CPU/GPU
2. Try different browser
3. Disable some animations in style.css
4. Update graphics drivers
```

### Dark mode not saving?
```
Solution:
1. Check if browser allows Local Storage
2. Check browser console for errors (F12)
3. Try in different browser
4. Clear browsing data and try again
```
